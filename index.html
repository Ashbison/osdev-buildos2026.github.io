PMM & VMM: Physical Memory Allocator

[Kernel] [Memory Mgmt]

Bump allocator + bitmap-based PMM for early kernel heap. Recursive page mapping for x86_64's 4-level paging. We implement a simple frame allocator and a virtual memory subsystem supporting page faults and demand paging foundations.

static uint8_t* bitmap;
static size_t total_frames;

void pmm_init(uint64_t mmap_addr, size_t mem_size) {
    total_frames = mem_size / FRAME_SIZE;
    size_t bitmap_size = BITMAP_SIZE(total_frames);
    bitmap = (uint8_t*) HIGHER_HALF_START;
    // mark reserved regions using e820/multiboot mmap
    for (size_t i = 0; i < total_frames; i++)
        bitmap_set(i, 1);       // all reserved initially
    // free usable RAM regions from boot info
    for (each_usable_region) {
        for (frame = base; frame < end; frame += FRAME_SIZE)
            bitmap_clear(frame / FRAME_SIZE);
    }
}

uintptr_t pmm_alloc_frame(void) {
    for (size_t i = 0; i < total_frames; i++) {
        if (!bitmap_test(i)) {
            bitmap_set(i, 1);
            return i * FRAME_SIZE;
        }
    }
    return 0;   // OOM
}
